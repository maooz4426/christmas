# christmas

デスクトップ画面に雪を降らせるアプリケーションです。

雪を降らせながら奥にあるターミナル操作等を行うことができます。

![snowflake](https://github.com/user-attachments/assets/6caf3c89-5641-4722-a43d-57a1ab63b4fa)



# 技術スタック

- Tauri
- Vite
- React
- pnpm

# インストール

アプリのインストールはこちらから


# ローカルで起動する場合

- 依存関係をインストール
```
pnpm install
```

- dev環境を起動
```
pnpm tauri dev
```

- ビルド
```
pnpm tauri build
```
