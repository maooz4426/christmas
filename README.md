# christmas

デスクトップ画面に雪を降らせるアプリケーションです。

雪を降らせながら奥にあるターミナル操作等を行うことができます。

# 技術スタック

- Tauri
- Vite
- React
- pnpm

アプリのインストールはこちらから


# ローカルで起動する場合

- 依存関係をインストール
```
pnpm install
```

- dev環境を起動
```
pnpm tauri dev
```

- ビルド
```
pnpm tauri build
```
